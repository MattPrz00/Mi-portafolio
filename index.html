Acerca de mí

¡Hola! Soy Matías

Desarrollador Web Full-Stack enfocado en la creación de aplicaciones funcionales, diseño responsivo y la resolución eficiente de problemas en equipo.

En el Front-End, construyo interfaces dinámicas y adaptables utilizando HTML, CSS, JavaScript y frameworks de estilos como Bootstrap.

En el Back-End, desarrollo lógica de servidores robusta con Python y Django, gestionando la persistencia de datos mediante bases de datos relacionales como MySQL y SQLite. Además, cuento con formación y conocimientos en la plataforma Low-Code GeneXus para el desarrollo ágil de aplicaciones corporativas.

Siempre en constante aprendizaje, busco proyectos desafiantes donde pueda aportar valor técnico y seguir potenciando mis habilidades tanto individuales como grupales.
